Getting Started

First server workflow

These steps match the current desktop workflow. AnxOS does not install unsupported runtimes or approve remote access without your action.

1

Download AnxOS Control Center for Windows or Linux.

2

Install and launch the desktop application.

3

Start the local AnxOS Agent or connect a remote Agent.

4

Add or select the Node you want to manage.

5

Open Prepare Node to verify Java, Docker, SteamCMD, Node.js, Python, and other required runtimes.

6

Install your first Marketplace server after dependency checks pass.

7

Monitor the server through Operations, Diagnostics, Files, and Node Health.

Install Guide

Install in a few steps

1

Download the installer for your platform.

2

Run the Windows setup, install the `.deb`, or mark the AppImage executable.

3

Open AnxOS, connect your remote node, then use Marketplace, Instances, Files, Console, and Stats.
