# AnxOS Control Center Version 1.7 Build 155

## Who it is for

Private Alpha users managing dedicated servers from Windows with a Debian Agent.

## New installations

Install the signed Windows package and complete first-run pairing with the Agent.

## Existing remote Agent users

Existing node registrations and pairing configuration are preserved during upgrade.

## Windows-only limitation

The packaged desktop release targets Windows for the local application host.

macOS Local Agent support is not documented or claimed.

## Upgrade guidance

Install Build 155 over the existing application. Confirm the displayed build and reconnect the selected Agent.

## Repair guidance

Use Agent Control repair and pairing recovery if the Agent is offline or credentials are stale.

## Validation

This candidate includes the validated Public Access, SteamCMD update, Windows Agent lifecycle, and paired-Agent status fixes. Complete real-machine Windows installation and real-machine Windows installation acceptance before public release.
