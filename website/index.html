Features

Built for server work

Minecraft server management

Create, monitor, start, stop, and inspect Minecraft instances from one desktop workspace.

Modpack marketplace installs

Browse provider packs and install CurseForge or Modrinth servers with guided manual-download recovery.

Remote Debian agent support

Connect the app to a Debian node for controlled file, command, and service operations.

Logs, files, commands, and stats

Work inside logs, file tools, command panels, server stats, and runtime views without changing apps.

Auto-update support

Use Check for update to pull the newest GitHub release when a new build is published.

Install Guide

Install in a few steps

1

Download the installer for your platform.

2

Run the Windows setup, install the `.deb`, or mark the AppImage executable.

3

Open AnxOS, connect your remote node, then use Marketplace, Instances, Files, Console, and Stats.

Safety / SmartScreen

About first-run warnings

Windows SmartScreen may warn on new releases because AnxOS is distributed directly through GitHub. Download only from the official repository release page and verify the file name matches the release shown here.
