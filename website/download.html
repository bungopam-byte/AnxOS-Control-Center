System requirements

  • Windows 10 or newer, x64, for the Windows setup and portable builds.
  • Linux x64 distributions that support AppImage, or Debian/Ubuntu-compatible systems for `.deb` packages.
  • Network access to your local or remote AnxOS Agent for remote node features.
  • Administrator privileges may be required for service setup, runtime installation, and firewall changes.

Install guidance

  • Windows: download the setup executable, run it, then launch AnxOS Control Center.
  • Linux AppImage: mark the file executable before launching it.
  • Linux `.deb`: install with your package manager, then start AnxOS Control Center from the app menu.
  • Alpha software: keep backups of important server data before using destructive operations.
