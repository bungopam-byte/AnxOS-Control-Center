System requirements

  • Windows 10 or Windows 11 on a 64-bit processor.
  • At least 4 GB RAM and 2 GB free storage for the app and Local Agent; servers and backups need additional space.
  • Internet access for downloads, updates, templates, and dependency installers.
  • Administrator approval is required when installing the Local Agent service or managed dependencies.

Install guidance

  • Choose Download AnxOS for Windows unless you specifically need the portable build.
  • Open the installer, launch AnxOS Control Center, then choose Use This PC during onboarding.
  • The Local Agent setup runs inside AnxOS and does not require Node.js, terminal commands, JSON editing, or token copying.
  • If Windows SmartScreen appears on an unsigned private-alpha build, verify the filename and release notes before continuing.
