Windows installation

Install AnxOS and Prepare This PC

This guide follows the normal Windows setup flow. You should not need terminal commands, manual JSON edits, Node.js installation, or Agent token copying.

1

Download the Windows installer from the Download page.

2

Open the installer after the download finishes.

3

If Windows shows a SmartScreen warning for an unsigned private-alpha build, verify the AnxOS filename and choose to continue only if it matches the release notes.

4

Launch AnxOS Control Center from the Start menu or desktop shortcut.

5

Complete onboarding and choose Use This PC.

6

Install the Local Agent from inside AnxOS.

7

Allow administrator permission when Windows asks to install the Local Agent service.

8

Let AnxOS scan dependencies such as Java, Git, SteamCMD, Docker, FFmpeg, and tunnel providers.

9

Install required dependencies through AnxOS when a selected Marketplace template needs them.

10

Open Marketplace.

11

Create your first server, then use Instances, Files, Backups, Console, and Public Access from This PC.

Troubleshooting

Common Windows setup issues

Windows SmartScreen warning

Private-alpha builds may be unsigned. Confirm the installer name, version, and release notes before continuing.

Local Agent installation failure

Use Retry or Repair in AnxOS. If antivirus software quarantined files, restore the official installer or Agent files before retrying.

Administrator permission denied

Sign in to a Windows administrator account or ask the computer owner to approve the Local Agent service installation.

Agent offline

Open Agent diagnostics in AnxOS and restart or repair the Local Agent service. The desktop app can run while the service starts.

Port already in use

Close the other app using the port or choose a different server port before installing or starting the server.

Antivirus quarantine

Only restore files when they came from the official AnxOS release. Then run Local Agent repair so AnxOS can verify the installation.

Docker restart required

Finish the Docker Desktop setup or restart Windows when Docker asks before scanning dependencies again.

Windows restart required

Restart Windows when an installer or driver requests it, then open AnxOS and retry the setup step.
